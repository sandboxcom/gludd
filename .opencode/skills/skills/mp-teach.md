# Teach

Teach a new skill or concept using a stateful workspace.

Workspace files: MISSION.md, reference/*.html (cheat sheets), RESOURCES.md, learning-records/*.md (key insights), lessons/*.html (self-contained lessons), NOTES.md.

Philosophy: Knowledge (from high-trust resources) + Skills (interactive lessons) + Wisdom (community interaction). Always challenge 'just enough' (zone of proximal development).
