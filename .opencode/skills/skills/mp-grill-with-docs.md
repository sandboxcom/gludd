# Grill With Docs

Grilling session that challenges your plan against the existing domain model.

- Interview the user relentlessly, one question at a time
- If a question can be answered by exploring the codebase, explore instead
- Challenge against CONTEXT.md glossary, sharpen fuzzy language
- Discuss concrete scenarios, cross-reference with code
- Update CONTEXT.md inline (glossary only, no implementation details)
- Offer ADRs sparingly (only when hard to reverse + surprising without context + real trade-off)
