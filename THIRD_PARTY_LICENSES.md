## Python Dependencies

A complete list of Python dependencies and their licenses is available in
the CycloneDX SBOM generated by `make sbom` and included in the distribution
tarball as `sbom.json`.

Key runtime dependencies:
- FastAPI (MIT) — web framework
- SQLAlchemy (MIT) — database ORM
- Ansible Core (GPL-3.0) — automation engine
- LangChain (MIT) — AI model integration
- Pydantic (MIT) — data validation

- Prometheus Client (Apache-2.0) — metrics
- Gunicorn (MIT) — WSGI server
- hvac (Apache-2.0) — HashiCorp Vault client

## Collected Prompts

The `config/prompt_profiles/collected/` directory contains system prompts
fetched from open-source projects. Each file includes:
- Source URL and repository
- License (typically Apache-2.0 or MIT)
- Retrieval date

Sources include: aider, OpenHands, SWE-agent, Cline.
